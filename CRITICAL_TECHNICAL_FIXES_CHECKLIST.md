## TROUBLESHOOTING

### Issue: Build fails after deleting sitemap.xml.js
**Solution:**
- Verify sitemap.xml.js deleted cleanly
- Check no other files reference it
- Run: `npm run build` to regenerate

### Issue: Robots.txt still shows old domain
**Solution:**
- Clear browser cache: Ctrl+Shift+Del
- Hard refresh: Ctrl+Shift+R
- Check robots.txt file directly (not cache)

### Issue: Sitemap.xml not generating
**Solution:**
- Verify astro.config.mjs has `site:` property
- Check Astro version compatible with auto-sitemap
- Run: `npm run build` with verbose logging

### Issue: Dates not updating in live site
**Solution:**
- Clear Netlify cache
- Deploy again
- Wait 24 hours for re-crawl
